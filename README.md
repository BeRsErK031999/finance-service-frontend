# finance-service-frontend

Baseline frontend shell for the `financeService` backend.

## Stack

- React 19
- TypeScript
- Vite
- React Router
- TanStack Query
- React Hook Form
- Zod
- MUI
- Axios

## Commands

```bash
yarn
yarn dev
yarn build
yarn lint
yarn type-check
```

## Environment

Create `.env` from `.env.example` and set:

```bash
VITE_API_BASE_URL=http://localhost:3000
```
